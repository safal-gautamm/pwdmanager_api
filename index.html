Password Manager API

Use the following endpoints to manage your accounts and passwords.

Create Account

Creates a new user account and returns an API key.

POST /create?name=YOUR_NAME

On Terminal:

curl -X POST "http://localhost:5555/create?name=John"

Add Password

Stores a new site password or updates it if already exists.

POST /add?apikey=API_KEY&site=SITE&username=USER&password=PASS

On Terminal:

curl -X POST "http://localhost:5555/add?apikey=ABC123&site=github.com&username=john&password=12345"

Update Site

Updates username or password (or both) for an existing site.

PUT /update?apikey=API_KEY&site=SITE&username=USER&password=PASS

On Terminal:

curl -X PUT "http://localhost:5555/update?apikey=ABC123&site=github.com&password=newpass"

View Passwords

Returns all saved passwords for a user (decrypted).

GET /view?name=NAME&apikey=API_KEY

On Terminal:

curl -X GET "http://localhost:5555/view?name=John&apikey=ABC123"

Delete All Passwords

Deletes all stored passwords for a user.

DELETE /delete?name=NAME&apikey=API_KEY

On Terminal:

curl -X DELETE "http://localhost:5555/delete?name=John&apikey=ABC123"
