Welcome to Password Manager API

Use the following endpoints to manage your passwords:

Create Account

POST /create?name=YOUR_NAME

Add Password

POST /add?apikey=YOUR_APIKEY&site=SITE_NAME&username=USERNAME&password=PASSWORD

View Passwords

GET /view?name=YOUR_NAME&apikey=YOUR_APIKEY

Delete All Passwords

DELETE /delete?name=YOUR_NAME&apikey=YOUR_API_KEY

Replace YOUR_NAME, YOUR_APIKEY, SITE_NAME, etc., accordingly.
